## ⚠️ Pitfalls
<details>
<summary><b>❌ Microphone not detected</b></summary>

- Ensure the mic is plugged in and set as default in Windows settings.
- Check privacy settings: `Settings > Privacy > Microphone` (enable access).
- Restart the assistant after changing mic settings.
</details>

<details>
<summary><b>❌ High CPU usage</b></summary>

- Close background apps to free up resources.
- Lower the `Audio Sample Rate` in settings (default: 16kHz).
- Disable unused voice commands in the config.
</details>

<details>
<summary><b>❌ False positives in hotword detection</b></summary>

- Avoid common words (e.g., "Hey" alone).
- Increase the `Confidence Threshold` in settings (range: 0.7–0.95).
- Train the model with your voice (Settings > Voice Training).
</details>

<details>
<summary><b>❌ Dictation not working in [App]</b></summary>

- Some apps (e.g., games) block virtual keyboards.
- Use `Windowed Mode` or `Borderless Window` for better compatibility.
- Try running the assistant as Administrator.
</details>
